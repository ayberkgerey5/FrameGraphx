## Controls
`W/S` - move forward/backward
`D/A` - move right/left
`V/C` - move up/down
`arrows` - rotation
`[` - next shader
`]` - previous shader
`R` - reload shader
`T` - reset time
`F` - freeze time
`U` - run shader debugger for pixel under cursor
`space` - pause rendering
`escape` - exit
`left mouse button` - rotate by mouse
